
# iOSTaskApp

A simple iOS appplication that user can sign up and login to app and create tasks to follow using



## Frontend

- UIKit

## Backend

- Swift 5

## Third Party Libraries

- Firebase
- Loaf
- FSCalendar
- MBProgressHUD
